Cache Simulator


          void run_cpu(CPU* cpu) {
            TraceLine trace_line;
            while (read_address(cpu, &trace_line) != EOF) {
              cpu->address_count++;
              AccessResult result = cache_access(cpu->cache, &trace_line);
              if (result == HIT) {
                cpu->hits++;
              }
              else if (result == COLD_MISS) {
                cpu->cold++;
              }   
              else {
                cpu->conflict++;
              }
            }
          }

Constructed the cache data structures. Then read input from supplied trace files, extracting relevant information from the inputted memory addresses using bitwise operation. The LRU cache replacement algorithm was implemented to store most recently read trace details in the cache. Written in C.
